# Installation

### Download or clone the project files
(from github)

### Virtual environment

Create the virtual environment. In console:
```
$ python3 -m venv env
```
Activate the virtual environment:
```
$ source env/bin/activate
```

Install the dependencies:

```
$ pip install -r requirements.txt
```

### run the server

```
$ ./manage.py runserver
```
